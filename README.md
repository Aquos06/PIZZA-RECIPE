# LAB_303_披薩食譜

食材：

    1. 麵粉 240g
    2. 水 200cc 
    3. 植物油2 tablespoon 
    4. 可果美番茄糊 3 tablespoon 
    5. 自選香料 3灑 
    6. 起司適量 
    7. 鹽巴3g 


作法：

    1. 將食材放在料理台
    2. 攪拌麵粉、植物油、水、鹽
    3. 搓揉均勻麵糰
    4. 蓋上濕濕的布
    5. 攤平
    6. 自己做披薩醬，調味番茄糊
    7. 將披薩醬塗到披薩皮
    8. 加上起司
    9. 整理一下外觀，確定披薩的形狀
    10. 放進烤箱攝氏160度16分鐘
